# Security Policy

## Supported versions

正式发布后，`0.1.x` 是当前受支持的版本线。未发布的开发提交不提供安全支持承诺。

## Reporting a vulnerability

请不要在公开 Issue 中披露漏洞、凭据或真实部署信息。使用 GitHub 的 [Private vulnerability reporting](https://github.com/Willxup/flowlens/security/advisories/new) 私下提交报告，并包含受影响版本、复现条件、影响和建议修复方向。

FlowLens 不会要求你提供真实 Clash Secret、登录密钥、Cookie 或数据库。请使用最小化、脱敏后的复现材料。
